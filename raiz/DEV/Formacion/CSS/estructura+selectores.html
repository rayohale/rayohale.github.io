Estructura + Selectores

ESTRUCTURA


::: Empezamos aprendiendo a ‘enlazar’ (un archivo CSS) ::: CSS: “Frontend”

::: La estructura es simple:

selector {
propiedad: valor;
}

::: CUATRO ESTILOS: Estilo en línea, estilos definidos por una etiqueta (style), externo (link), import
::: las que más se usan son: con un archivo externo, y en linea

::: Archivo externo: archivo MODULARIZADO (podemos separar el código), no uno tipo “spaguetti” (anti patrón de diseño)

::: en HTML TODOS son cajas, dos tipos de cajas, las que completan la linea (display block) y las que NO (display in-line). Elementos en bloque y elementos en línea

::: tipos de SELECTORES: Univesal; de tipo ; clases ; ID ; por atributo ; descendiente ; pseudo-clases

::: Estructura de HTML
