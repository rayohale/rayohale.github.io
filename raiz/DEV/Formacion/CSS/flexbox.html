Flexbox

Introducción

  • * Flexbox: cajas flexibles. (antes se hacía todo con tablas)
  • * Requiere de dos cosas, un flex container (el que contiene este o estos items) y un flex item.
  • * si damos display flex a un contenedor, de por si se va a comportar como un bloque.. en donde vamos a notar los cambios es adentro del cntenedor
  • * En el eje cartesiano (x-horizontal-,y-vertical-) -> main axis (x) / cross axis (y)
  • - main start (left) + main end (right) = main axis
  • - cross start (up) + cross end (down) = cross axis
  • * lo que hacemos con CSS es cambiar el flujo de MA y CA; flexbox invierte este orden
