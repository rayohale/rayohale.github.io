PROPIEDAD DISPLAY:

modifica el comportamiento de las cajas, ya no la interacción tanto de las cajas.. hablamos de relaciones.
- Tiene muchos valores
- INLINE BLOCK: le podemos dar width y height, bloque que se comporta como line, se ajusta al contenido
- en HTML ya casi no se usan TABLES, se reemplazaron por otras cosas
- las que más se usan son BLOCK INLINE INLINE-BLOCK GRID FLEX INLINE-FLEX INLINE-GRID

TRUQUITO!:
cntrl+F
cntrl*H
- reemplazar todos por X

OVERFLOW (lo que sobra) (es un shorthand)
"inherit" (que se sobresalga nomas)
"auto" en caso de que el contenido sobrepase la caja, podemos scrolear en la caja
"scroll" obliga a que la caja esté scrolleada 00:34
"hidden" oculta, si una imagen está corrida, que sobresale, se le asigna ese valor, se oculta
